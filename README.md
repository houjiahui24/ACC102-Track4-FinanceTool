# ACC102 Track4 - Interactive Financial Dashboard

## Project Overview
This project is an interactive financial analysis platform built with Streamlit. It visualizes key financial ratios and supports dynamic filtering by company and year.

## Data Source
- Database: WRDS Compustat
- Data Type: Annual Financial Data
- Period: 2020 – 2024
- Companies: AAPL, MSFT, NVDA, GOOGL

## Visualizations (5 Charts)
1. ROE Trend Line Chart
2. Annual Revenue Bar Chart
3. Net Profit Margin Trend
4. Leverage Level Area Chart
5. Comprehensive Financial Radar Chart

## How to Run
1. Install dependencies:
   pip install -r requirements.txt
2. Start the application:
   streamlit run app.py

## Notes
- Data is pre-downloaded from WRDS for stable display.
- The program runs smoothly both locally and on Streamlit Cloud.
- No WRDS account is required to view results.
